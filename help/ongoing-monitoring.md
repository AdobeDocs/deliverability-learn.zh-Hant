# 持續監控

以下提供一些方法，可協助識別您可能需要專家支援的可能問題：

* 在硬或軟跳出中觀察到尖峰。 這可能表示封鎖清單或其他傳遞能力問題。
* 在傳遞率保持高位的情況下，開啟和點按量度有明顯下降。 這表示可能會大量放置資料夾。
* 投訴大幅增加。 這可能是由於清單來源品質不佳所導致。
* 您有任何可能影響傳遞能力的策略性方案。 這些包括但不限於訂閱者贏取、參與策略、季節性策略或頻率重大變更，以及行銷活動型別。
