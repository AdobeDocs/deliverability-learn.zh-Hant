# 持續監控

以下提供一些方法，可協助您識別可能需要專家支援的可能問題：

* 硬跳出或軟跳出出現觀察到的尖峰。 這可能表示封鎖清單或其他傳遞能力問題。
* 傳遞率保持高位時，開啟和點按量度明顯減少。 這表示可能有大量資料夾的位置。
* 投訴大幅增加。 這可能是由於品質不佳的清單來源所造成。
* 您有任何可能會影響傳遞能力的策略性方案。 這些包括但不限於訂閱者贏取、參與策略、季節性策略或頻率的重大變更，以及促銷活動型別。
