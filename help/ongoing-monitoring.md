# 持續監控

以下是一些方法，可協助您找出可能需要專家支援的問題：

* 在硬彈回數或軟彈回數中，有觀察到尖峰。 這可能表示封鎖清單或其他交付性問題。
* 開啟和點按量度明顯減少，但傳送率仍維持高位。 這表示可能大量放置資料夾。
* 投訴大幅增加。 這可能是由品質不佳的清單來源所造成。
* 您有任何可能影響交付能力的策略計畫。 這些功能包括但不限於訂閱者獲取、參與策略、季節性策略或頻率的重大變更，以及宣傳類型。
