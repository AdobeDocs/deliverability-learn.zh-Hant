# 最佳化傳遞能力的內容最佳實務

內容是關鍵。 您已閱讀我們對於相關性的觀點，但以下是最佳化內容傳遞能力的幾個額外秘訣：

* **避免HTML檔案太大**。 保持在100KB以下，但儘量保持在60到80KB之間，以防止傳送速度緩慢。
* **使用alt標籤對您有利**。 Alt標籤會在HTML的影像程式碼中即時顯示，如果影像不可見或無法載入，則會顯示文字。 與其使用簡單的說明，如「產品快照」，您可能會想要說一些更吸引人的話，如「立即購買可享受30%優惠」。
* **避免太多影像。** 根據預設，大部分的ISP現在會封鎖影像。 您想要有一種方式，可以在未啟用影像的情況下擷取您的對象，讓他們接著啟用影像。
