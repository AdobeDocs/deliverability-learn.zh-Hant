# 付諸實踐

本指南已說明傳遞能力的細微差別，並遵循最佳實務。

成功有四個關鍵支柱：

1. 在註冊期間設定適當的期望值，並具備防止地址錯誤的良好註冊過程。
2. 提供相關且及時的內容。
3. 通過刪除變壞的地址來維護清單。
4. 隨時監控、測試和調整。

如果您不清楚或需要問題方面的協助，請聯絡您的Adobe傳遞能力顧問或傳遞能力專家尋求協助。
